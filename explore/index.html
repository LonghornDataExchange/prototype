Use

We envision you using the provided information in several ways:

  • Options: Discover what range of options are available to you that other organizations are using.

  • Examples: Find examples of events other student organizations have executed to inspire your own events.

  • Experts: Find organizations that have already done what you are trying to do using our events reporting.

  • Trends: Get ahead of university-wide trends and changes by looking through our analysis.

Views

You can view information in three different forms: aggregated views, event views, or analysis.

Aggregate Views

Aggregate views provide information grouped by some characteristic. This view groups organization's experiences by a category, giving you a clean and straightforward way to analyze a range of options. This view also presents the many different experiences of various reporters about a single subject in a single place.

Aggregate views are a great place to start if you are brainstorming, looking for options, or trying to see what's out there. By browsing through the aggregate view, you will see the best source of unique options which other organizations have already implemented.

Please note that some information does not have a aggregate view because the data is inherently unique or averaged before it is entered. In these cases, there will be no aggregate view present.

Event Views

Most the information we receive is event focused, which means that it contains information only about a single organization at a single point in time. We treat this as the raw data behind the aggregate views we present.

This information is best used to compare your specific events or plans with another organization. For example, if you want to copy a social from another organization, you might want to see more details about the specific event that the organization reported.
